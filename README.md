# Blogmanager

유튜브 차트 분석 영상을 네이버 블로그 글로 자동 변환·발행하는 Windows 프로그램.

## 다운로드

➡️ **[최신 버전 다운로드 (Releases)](https://github.com/projovermind/blogmanager-releases/releases/latest)**

`Blogmanager.exe` 파일을 받아서 실행하세요.

## ⚠️ 실행 전 필독

- **OneDrive / 클라우드 동기화 폴더에 두지 마세요.** `C:\Blogmanager\` 같은
  일반 폴더에 받아서 실행하세요. (onefile 실행 파일이 클라우드 placeholder로
  비워지면 실행이 실패합니다.)
- 첫 실행 시 Chromium·ffmpeg를 자동으로 내려받습니다 (약 250MB, 1회만).
- 계정 정보는 관리자 시트에서 자동 적용됩니다.

## 버전

최신 버전 및 변경 내역은 [Releases](https://github.com/projovermind/blogmanager-releases/releases) 참고.
